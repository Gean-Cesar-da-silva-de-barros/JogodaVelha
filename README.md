# 🎮 Jogo da Velha

Um projeto simples e interativo de **Jogo da Velha (Tic-Tac-Toe)** desenvolvido com HTML, CSS e JavaScript.

## 🚀 Demonstração

👉 https://gean-cesar-da-silva-de-barros.github.io/JogodaVelha/

## 🧠 Funcionalidades

* Alternância automática entre jogadores (X e O)
* Verificação de vitória
* Detecção de empate
* Botão de reset para reiniciar o jogo
* Interface simples e intuitiva

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 📁 Estrutura do projeto

```
📦 jogodavelha
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┗ 📄 script.js
```

## 🎯 Objetivo

Este projeto foi desenvolvido com foco em aprendizado prático de lógica de programação e manipulação do DOM.
